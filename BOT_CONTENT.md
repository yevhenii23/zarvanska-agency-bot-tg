## Що потрібно надати для повного запуску

### 1. Google Sheets
- Створити проект в Google Cloud Console
- Увімкнути Google Sheets API
- Створити Service Account → скачати JSON-ключ
- Створити Google таблицю → дати доступ (Editor) email-у з service account
- Надати: **JSON файл credentials** + **ID таблиці**

### 2. Посилання на сайт
- URL сайту → `.env` → `SITE_URL=...`

### 3. Посилання на календар (опціонально)
- Для сценарію "Хочу консультацію" — Calendly / Cal.com

### 4. Хостинг
Бот працює тільки поки запущений. Для 24/7 потрібен хостинг:

| Варіант | Ціна | Складність |
|---------|------|-----------|
| Railway.app | ~$5/міс | Дуже просто |
| Render.com | Free tier | Просто, free засинає |
| VPS (DigitalOcean/Hetzner) | $4-6/міс | Треба налаштувати |
| Fly.io | Free tier | Середня |
